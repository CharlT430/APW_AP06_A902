Sistemas Operativos

Competencia:

Desarrollar soluciones tecnológicas para entornos Web mediante fundamentos de programación orientada a objetos, base de datos y redes de área local que atiendan las necesidades de las organizaciones.

Objetivo de aprendizaje:

El alumno realizará la instalación y configuración de sistemas operativos y los servicios necesarios para la publicación de sitios Web.

Unidades de aprendizaje:

  1. Introducción a Sistemas Operativos
  2. Instalación de sistemas operativos basados en UNIX
  3. Comandos básicos de sistemas operativos basados en UNIX
  4. Administración básica del sistema operativo
  5. Instalación de servicios para aplicaciones Web

Unidades de aprendizaje:

I. Introducción a Sistemas Operativos

Objetivo de la unidad:

El alumno evaluará las ventajas del uso de licencias de código abierto.

  1. Describir los antecedentes de los sistemas operativos
  2. Establecer las características de las licencias de software libre y propietarios.
  3. Describir las características y funcionalidad del sistema operativo basado en UNIX

II. Instalación de sistemas operativos basados en UNIX

Objetivo de la unidad:

El alumno realizará la instalación de un sistema operativo basado en UNIX para probar su configuración en equipos de cómputo.

  1. Seleccionar sistemas operativos basados en UNIX en función de los requerimientos de instalación.
  2. Establecer las particiones en la instalación de sistemas operativos basados en UNIX.
  3. Seleccionar el formato y los sistemas de archivos de la instalación de sistemas operativos basados en UNIX.
  4. Realizar la instalación sistemas operativos basados en UNIX.

III. Comandos básicos de sistemas operativos basados en UNIX

Objetivo de la unidad:

El alumno administrará archivos, directorios y tarjetas de red de los sistema operativo basado en UNIX

  1. Realizar diversas tareas a partir del uso de comandos básicos de sistemas operativos basados en UNIX.
  2. Realizar el proceso de creación, modificación y eliminación de archivos y directorios, así como sus permisos.
  3. Establecer repositorios en sistemas operativos basados en UNIX
  4. Establecer la configuración de la tarjeta de red en sistemas operativos basados en UNIX.

    5. IV. Administración básica del sistema operativo

    Objetivo de la unidad:

    El alumno administrará un sistema operativo basado en UNIX mediante el uso de comandos para optimizar su rendimiento.

    1. Administrar cuentas de usuario y grupos en sistemas operativos basados en UNIX.
    2. Gestionar paquetes y procesos en sistemas operativos basados en UNIX.
    3. Programar scripts en Shell

      4. V. Instalación de servicios para aplicaciones Web

      Objetivo de la unidad:

      El alumno instalará los servicios Web, base de datos y de transferencia de archivos para la publicación de sitios Web.

      1. Establecer servicios Web en sistemas operativos basados en UNIX.
      2. Establecer servicios de gestión de base de datos en sistemas operativos basados en UNIX.
      3. Establecer servicios de transferencia de archivos en sistemas operativos basados en UNIX.
