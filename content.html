Aplicaciones Web

Competencia:

Desarrollar soluciones tecnológicas para entornos Web mediante fundamentos de programación orientada a objetos, base de datos y redes de área local que atiendan las necesidades de las organizaciones.

Objetivo de aprendizaje:

El alumno desarrollará aplicaciones Web con acceso a base de datos a través de lenguajes y herramientas especializadas para distribuirlas en internet.

Unidades de aprendizaje:

  1. Ambientes de desarrollo Web
  2. Desarrollo de Frontend
  3. Desarrollo de Backend
  4. Publicación de aplicaciones de Backend

I. Ambientes de desarrollo Web:

Objetivo de la unidad:

El alumno configurará el ambiente de desarrollo para codificar aplicaciones Web.

  1. Identificar conceptos de aplicaciones web
  2. Identificar ventajas y desventajas de uso de aplicaciones web del lado del cliente y servidor
  3. Realizar la instalación y configuración de ambiente de desarrollo web

II. Desarrollo de Front End:

Objetivo de la unidad:

El alumno desarrollará aplicaciones Web dinámicas utilizando un lenguaje El alumno implementará estándares de diseño Web para crear interfaces responsivas.

  1. Realizar documentos HTML5
  2. Realizar formularios HTML y HTML5
  3. Diseñar la presentación de documentos HTML en hojas de estilo en cascada
  4. Desarrollar sitios Web responsivos en frameworks.

III. Desarrollo de Back End:

Objetivo de la unidad:

El alumno desarrollará aplicaciones Web dinámicas utilizando un lenguaje de programación orientado a objetos del lado del servidor.

  1. dentificar la sintaxis del lenguaje de programación del lado del servidor
  2. Métodos de comunicación entre cliente y servidor: get y post.
  3. Definir el proceso de conexión, inserción, modificación, eliminación y consulta de Bases de Datos desde aplicaciones Web.
  4. Identificar el proceso de carga y descarga de archivos almacenados en el servidor.
  5. Desarrollar Aplicaciones Web que implementen el uso de sesiones.

    6. IV. Publicación de Aplicaciones:

    Objetivo de la unidad:

    El alumno publicará aplicaciones Web para su utilización en redes locales y en línea.

    1. Seleccionar el servicio de Hosting de acuerdo a las necesidades de la aplicación Web.
    2. Seleccionar el protocolo y la herramientas de transferencia de archivos de acuerdo a la necesidades de la aplicación Web.
    3. Realizar la publicación de aplicaciones Web de forma local y remota de acuerdo al servicio seleccionado.
